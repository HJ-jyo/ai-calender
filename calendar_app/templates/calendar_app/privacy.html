プライバシーポリシー

Smart Calendar（以下「本アプリ」）は、利用者のプライバシーを尊重し、個人情報の保護に努めます。本ポリシーは、本アプリが Google API から取得する情報の取り扱いについて説明するものです。

1. 取得する情報の種類と利用目的

本アプリは、Google OAuth 認証を通じて以下の情報にアクセスします。

  • Google カレンダーの参照・編集権限: 利用者がアップロードした画像（レシートやメモ）から AI が解析した予定を、利用者のカレンダーに自動登録するためにのみ使用します。
  • メールアドレス: 利用者の識別および認証のために使用します。

2. データの利用方法（AI 解析について）

本アプリは、Google Gemini API を使用して画像を解析します。アップロードされた画像データは、解析処理のために一時的に処理されますが、本アプリのサーバー内に永続的に保存されることはありません。

3. データの共有と第三者提供

本アプリは、取得したデータを第三者に販売または共有することはありません。取得したデータは、カレンダー登録という本アプリの核となる機能を提供するためだけに利用されます。

4. データの保持と削除

利用者は、Google アカウントの設定からいつでも本アプリへのアクセス権限を取り消すことができます。本アプリ側で保持しているセッション情報などは、利用者の要求に応じて削除可能です。
